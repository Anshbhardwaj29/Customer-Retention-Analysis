# Customer-Retention-Analysis
# Project 2: Customer Retention and RFM Segmentation Analysis

## 🎯 Objective
Analyzed customer behavioral data using Python to identify churn patterns, evaluate customer lifetime value trajectories, and perform cohort segmentation to optimize business retention strategies.

## 🛠️ Tech Stack & Skills Learned
- **Language:** Python
- **Libraries:** Pandas, NumPy, Seaborn, Matplotlib
- **Methodologies:** RFM Analysis (Recency, Frequency, Monetary), Cohort Segmentation, Correlation Mapping
- **Insights Extracted:** Customer lifetime value buckets, seasonal retention drop-offs, and churn risk indicators.

## 📊 Key Findings
1. **RFM Segmentation:** Grouped users into actionable categories (Champions, Loyal, At-Risk, Hibernating) using percentile-based quartile cuts (`pd.qcut`).
2. **Cohort Matrix:** Built a time-series user retention heatmap tracking active month-on-month engagement.
3. **Correlation Analysis:** Identified inverse correlation between Recency and Frequency, statistically proving that highly active buyers churn less.
